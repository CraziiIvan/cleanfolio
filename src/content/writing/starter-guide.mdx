### Installation

To get started with Cleanfolio, clone the repository and install the necessary dependencies.

```bash
git clone https://github.com/yourusername/cleanfolio.git
cd cleanfolio
npm install
```

### Folder Structure

```
cleanfolio/
├── public/
├── src/
│ ├── assets/
│ ├── icons/
│ ├── content/
│   ├── project/
│   └── writing/
│ ├── layouts/
│ ├── pages/
│ ├── components/
│ ├── lib/
│ ├── App.js
│ └── types.ts
├── .gitignore
├── package.json
└── README.md
```

- **src/**: Source code for components, pages, and styles.
- **assets/**: Assets like images and fonts.
- **layouts/**: Layout components.
- **pages/**: Page components (Home, Projects, Writings).
- **components/**: Reusable UI components.
- **content/**: Markdown files for content.
- **lib/**: Utility functions and constants.
- **pages/index.astro**: Entry point for the application.

### Updating Content

If you don't want to customize further, you can just update the content in the following files.

1. **Home Page**: Modify the content in `src/pages/index.astro` and `src/lib/constants.ts`.
2. **Projects Page**: Manage your project posts in `src/content/project/`.
3. **Writings Page**: Manage your blog posts in `src/content/writing/`.
3. **Content Photos**: Manage your content photos in `src/assets/`.

### Deploying Your Portfolio

To deploy your portfolio, you can use services like Netlify, Vercel, or GitHub Pages. Here’s a quick guide for deploying with Netlify:

1. **Build the project**:

```bash
npm run build
```

2. **Deploy**:
   - Log in to [Netlify](https://www.netlify.com/).
   - Create a new site from Git.
   - Connect your repository and deploy.
