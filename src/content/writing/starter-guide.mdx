### Installation

To get started with Cleanfolio, clone the repository and install the necessary dependencies.

```bash
git clone https://github.com/yourusername/cleanfolio.git
cd cleanfolio
npm install
```

### Folder Structure

```
cleanfolio/
├── public/
├── src/
│ ├── assets/
│ ├── icons/
│ ├── content/
│   ├── project/
│   └── writing/
│ ├── layouts/
│ ├── pages/
│ ├── components/
│ ├── lib/
│ ├── App.js
│ └── types.ts
├── .gitignore
├── package.json
└── README.md
```

- **public/**: Static files like images and the main HTML file.
- **src/**: Source code for components, pages, and styles.
- **components/**: Reusable UI components.
- **pages/**: Page components (Home, About, Portfolio, Blog, Contact).
- **styles/**: CSS and styling files.
- **App.js**: Main application component.
- **index.js**: Entry point for the application.

### Updateing Content

1. **Home Page**: Modify the content in `src/pages/Home.js`.
2. **About Page**: Update your bio and skills in `src/pages/About.js`.
3. **Portfolio Page**: Add your projects in `src/pages/Portfolio.js`.
4. **Blog Page**: Manage your blog posts in `src/pages/Blog.js`.
5. **Contact Page**: Update your contact information in `src/pages/Contact.js`.

### Deploying Your Portfolio

To deploy your portfolio, you can use services like Netlify, Vercel, or GitHub Pages. Here’s a quick guide for deploying with Netlify:

1. **Build the project**:

```bash
npm run build
```

2. **Deploy**:
   - Log in to [Netlify](https://www.netlify.com/).
   - Create a new site from Git.
   - Connect your repository and deploy.
